# Satellite Telemetry Simulator

A Python-based satellite telemetry simulator that generates simulated CubeSat telemetry packets.

## Project Goal

This project simulates basic satellite telemetry data such as:

- Timestamp
- Satellite ID
- Battery voltage
- Battery current
- Temperature
- Altitude
- Orbital speed
- Signal strength
- System status

The goal is to build a clean and progressively more advanced Python portfolio project related to space systems, telemetry, and aerospace software engineering.

## Current Features

- Generates simulated telemetry packets
- Uses realistic units for telemetry fields
- Displays telemetry data in a readable format
- Generates multiple telemetry packets in sequence
